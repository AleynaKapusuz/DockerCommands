# DockerCommands

## Install Container :
```
docker pull <containername>
```

## Building a Container :
```
docker run -d --name <name> -p 8080:80<name>
```

## See Running Containers :
```
docker container ls -a
```

## Deleteing a Container
```
docker rm <name>
```

## Show the Image 
```
docker images
```

## Running a Image
```
docker run <ImageID>
```
